## Часть 1: Теоретические основы — история и молекулярная биология

### 1.1 Краткая история

| Год | Событие |
|-----|---------|
| 1869 | Фридрих Мишер открывает нуклеин (ДНК) |
| 1909-1929 | Фебус Левен открывает нуклеотиды, рибозу, дезоксирибозу |
| 1950 | Эрвин Чаргафф — правила комплементарности (A=T, G=C) |
| 1953 | Уотсон, Крик, Франклин, Уилкинс — структура ДНК |
| 1977 | Максам-Гилберт, Сэнгер — методы секвенирования |
| 1983-1985 | Кэри Муллис — ПЦР |
| 2005+ | Следующее поколение (NGS) — Illumina, 454, SOLiD |
| 2010+ | Третье поколение — ONT, PacBio |

### 1.2 Структура ДНК (напоминание)

**Компоненты:** азотистые основания (A, T, G, C), дезоксирибоза, фосфатная группа

**Цепи:** антипараллельны (5'→3' и 3'→5')

**Синтез:** полимераза работает в направлении 5'→3', используя праймер

**РНК:** рибоза (с OH-группой) + уракил вместо тимина

### 1.3 Полимеразная цепная реакция (ПЦР)

**Этапы цикла:**

1. **Денатурация** (94-98°C) — разделение цепей
2. **Отжиг праймеров** (50-65°C) — гибридизация с матрицей
3. **Элонгация** (72°C) — полимераза достраивает цепи

**Результат:** экспоненциальное накопление (~2ⁿ после n циклов)

## Часть 2: Секвенирование первого поколения

### 2.1 Метод Максама–Гилберта (1977)

**Принцип:** химическая деградация ДНК по специфическим основаниям

**Этапы:**

1. Радиоактивная метка ³²P на 5'-конец
2. 4 пробирки с разными модификациями:
   - A+G — диметилсульфат
   - G — диметилсульфат + высокая температура
   - C+T — гидразин
   - C — гидразин + NaCl
3. Расщепление пиперидином
4. Электрофорез в полиакриламидном геле
5. Авторадиография

**Недостатки:** токсичные вещества, радиоактивность, сложность

### 2.2 Метод Сэнгера (дидезокси-терминация, 1977)

**Принцип:** ddNTP (дидезоксинуклеотиды) блокируют синтез ДНК — нет 3'-OH группы

#### Классическая версия

- 4 реакции, в каждой — один из ddNTP с радиоактивной меткой
- Получаются фрагменты разной длины, заканчивающиеся на определённый нуклеотид
- Электрофорез в 4 дорожках, авторадиография

#### Автоматизированная версия (с 1986)

- Флуоресцентные метки вместо радиоактивных
- Капиллярный электрофорез вместо геля
- Все 4 реакции в одной пробирке (разные красители)

#### Современное состояние (золотой стандарт)

| Параметр | Значение |
|----------|----------|
| **Точность** | 99.9% |
| **Длина прочтения** | 300-900 bp |
| **Чувствительность** | 15-20% (примеси) |
| **Стоимость** | Дешево |
| **Скорость** | Быстро |

**Применение:** плазмиды, валидация мутаций, небольшие фрагменты

## Часть 3: Секвенирование второго поколения (NGS) — общая схема

### 3.1 Основные этапы

#### 1. Приготовление библиотеки

- Выделение нуклеиновых кислот
- Фрагментация (если нужно)
- Добавление адаптеров (технические последовательности)

#### 2. Клональная амплификация

Создание множества копий одной молекулы в локальном кластере:
- Эмульсионная ПЦР
- Мостиковая амплификация
- Амплификация катящимся кольцом

#### 3. Секвенирование

Циклическое определение нуклеотидов:
- Пиросеквенирование
- Секвенирование синтезом
- Секвенирование лигированием

#### 4. Base calling и анализ

- Преобразование сигналов в последовательности (FASTQ)
- Демультиплексирование (разделение по образцам)

### 3.2 Почему РНК переводят в ДНК?

- РНК нестабильна (гидролизируется)
- ДНК более стабильна для хранения
- Полимеразы работают с ДНК-матрицей

## Часть 4: Клональная амплификация

### 4.1 Эмульсионная ПЦР (emulsion PCR)

**Платформы:** Roche 454, SOLiD, Ion Torrent

**Принцип:**

1. Создаётся водно-масляная эмульсия с микрокаплями
2. В каждой капле: магнитная гранула, одна молекула ДНК, праймеры, полимераза, dNTP
3. ПЦР внутри капель
4. На каждой грануле — копии одной исходной молекулы
5. Эмульсия разрушается, гранулы собираются

### 4.2 Мостиковая амплификация (Bridge amplification)

**Платформа:** Illumina

**Принцип:**

1. На проточной ячейке (flow cell) иммобилизованы олигонуклеотиды (адаптеры P5 и P7)
2. ДНК гибридизуется с олигонуклеотидами
3. ПЦР: достройка цепи → денатурация → отжиг на соседний олигонуклеотид (мостик)
4. Циклы повторяются
5. Кластеры из ~1000 копий одной молекулы

### 4.3 Амплификация катящимся кольцом (RCA)

**Платформа:** BGI/MGI (DNBSEQ)

**Принцип:**

1. ДНК-фрагменты с адаптерами циркуляризуются (замыкаются в кольцо)
2. Полимераза с высокой процессивностью обходит кольцо много раз
3. Образуется конкатемер (DNA nanoball)
4. Наношарики гибридизуются с паттернированной подложкой

**Преимущество:** ошибки не накапливаются (копирование всегда с оригинала)

## Часть 5: Платформы второго поколения (подробно)

### 5.1 Roche 454 (пиросеквенирование)

**Статус:** снят с производства

**Принцип:**

1. Поочередно добавляют dNTP
2. При встраивании выделяется пирофосфат (PPi)
3. PPi → АТФ (сульфурилаза)
4. АТФ → свет (люцифераза)
5. CCD-камера детектирует свет

**Проблема:** гомополимеры (5 vs 6 одинаковых букв трудно различить)

**Длина рида:** до 600 bp

### 5.2 SOLiD (Sequencing by Oligonucleotide Ligation)

**Статус:** снят с производства

**Принцип:** лигирование ди-нуклеотидных зондов с флуорофорами

**Плюсы:** высокая точность, низкая ошибка

**Минусы:** короткие прочтения, сложная обработка

### 5.3 Ion Torrent (Thermo Fisher)

**Статус:** активно используется (клиники, диагностика)

**Принцип:** детекция ионов H⁺, выделяющихся при встраивании нуклеотида

**Детекция:** CMOS-сенсор (как в фотокамерах) измеряет изменение pH

**Процесс:**

1. Нуклеотиды добавляются по очереди (T, A, C, G)
2. При встраивании выделяется H⁺ → pH падает → сигнал
3. Амплитуда пропорциональна числу встроенных нуклеотидов

**Плюсы:** дешевизна, скорость, нет оптики

**Минусы:** ошибки на гомополимерах

### 5.4 Illumina (Solexa) — лидер рынка (80-90%)

**Амплификация:** мостиковая ПЦР

#### Проточные ячейки

- **Patterned flow cell** — кластеры в лунках (меньше перекрывание)
- **Random flow cell** — кластеры случайно

#### Принцип SBS (Sequencing By Synthesis)

1. Добавляют все 4 меченых нуклеотида с обратимыми терминаторами
2. Каждый нуклеотид имеет уникальный флуорофор
3. Полимераза встраивает один нуклеотид
4. Терминатор блокирует дальнейшее встраивание
5. Сканирование лазерами в разных каналах
6. Отщепление терминатора и флуорофора
7. Цикл повторяется

#### Типы химии детекции

| Химия | Красители | Сканирования | Скорость | Точность |
|-------|-----------|--------------|----------|----------|
| 4-канальная | 4 | 4 на цикл | Медленно | Отличная |
| 2-канальная | 2 | 2 на цикл | Быстрее | Хорошая |
| 1-канальная | 1 | 2 на цикл | Очень быстро | Сложнее |

#### Парные прочтения (Paired-End)

- Read 1 — первый конец фрагмента
- Index — барcode для мультиплексирования
- Read 2 — второй конец фрагмента (после ресинтеза)

**Типичные длины:** 150+150, 250+250, 300+300

**Зачем?** Лучшее выравнивание, сборка геномов, структурные варианты

#### Производительность

| Модель | Производительность |
|--------|-------------------|
| MiniSeq | Низкая |
| MiSeq | Средняя |
| NextSeq | Высокая (~400 млн ридов) |
| NovaSeq | Ультравысокая (~16 млрд ридов = 16 Tb) |

### 5.5 BGI/MGI (DNBSEQ) — китайский конкурент

**Статус:** быстро набирает популярность, выиграла патентный иск

**Амплификация:** RCA (DNA nanoballs)

**Подложка:** паттернированная, 220 нм диаметр, 715 нм шаг

**Преимущества:**

- Меньше ошибок амплификации
- Меньше дупликатов ПЦР
- Высокая точность

#### CoolMPS — инновационная техника

- Немеченые нуклеотиды
- После встраивания — антитела с флуорофорами
- Сильный сигнал, меньше фона
- Выше точность

## Часть 6: Секвенирование третьего поколения (длинные прочтения)

### 6.1 Oxford Nanopore Technologies (ONT)

**Принцип:** ДНК протягивается через белковую нанопору (α-гемолизин) под напряжением

**Детекция:** ионный ток изменяется при прохождении каждого нуклеотида

**Устройства:**

| Модель | Назначение |
|--------|-----------|
| MinION | Карманный секвенатор (USB) |
| GridION | До 5 проточных ячеек |
| PromethION | Большие объёмы |

**Длина прочтений:** до 2 млн bp

**Качество:** ~90% точность на молекулу, но можно увеличить повторными чтениями (duplex)

**Плюсы:**

- Очень длинные прочтения (теломер-теломер сборки)
- Портативность
- Дешевизна
- Можно останавливать/продолжать

**Минусы:** высокая ошибка (вставки/делеции)

### 6.2 PacBio (Pacific Biosciences) — SMRT sequencing

**Принцип:** полимераза на дне наноямки (ZMW), меченые нуклеотиды

**Циркулярное секвенирование (CCS):**

1. ДНК зацикливается шпильками (SMRTbell)
2. Полимераза много раз обходит кольцо
3. Из нескольких прочтений строится консенсус (HiFi reads)

**HiFi reads:**

- Длина: 10-15 kb
- Качество: 99.9% (как Illumina!)

**Одинарные прочтения:**

- Длина: до 20-30 kb
- Качество: ~90%

**Плюсы:** высокая точность, длинные прочтения

**Минусы:** очень дорогое оборудование и реактивы

### 6.3 Сравнение поколений

| Параметр | 1-е (Сэнгер) | 2-е (Illumina) | 3-е (ONT) | 3-е (PacBio HiFi) |
|----------|-------------|---|---|---|
| **Длина рида** | 800 bp | 150-300 bp | До 2 Mb | 10-20 kb |
| **Количество** | 1 | Млрд | Млн | Млн |
| **Точность** | 99.9% | 99.9% | 90-98% | 99.9% |
| **Cost/Gb** | Высокая | Низкая | Очень низкая | Высокая |

## Часть 7: Формат FASTQ и качество прочтений

### 7.1 Структура FASTQ-файла

Каждое прочтение — 4 строки:

```
@EAS139:136:FC706VJ:2:2104:15343:197393 1:Y:18:ATCACG
GTGGTCATAGAGTTTATTTGGTTCCTTTGGTCTTAATTAGTTTGGATAAAATTTCTCTAATAGG
+
GGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGGG
```

| Строка | Описание |
|--------|---------|
| 1 | Заголовок (начинается с @) |
| 2 | Последовательность (A, C, G, T, N) |
| 3 | Разделитель (+) |
| 4 | Качество (Phred scores) |

### 7.2 Phred Quality Score (Q)

**Формула:** Q = −10 log₁₀ P, где P — вероятность ошибки

| Q | P | Точность |
|---|---|----------|
| 10 | 0.1 | 90% |
| 20 | 0.01 | 99% |
| 30 | 0.001 | 99.9% |
| 40 | 0.0001 | 99.99% |

**Интерпретация:**
- Q10 → 1 ошибка на 10 оснований
- Q20 → 1 на 100
- Q30 → 1 на 1000
- Q40 → 1 на 10000

### 7.3 Кодировка Phred в FASTQ

**Phred+33 (Illumina 1.8+):**
- Символ = chr(Q+33)
- Q от 0 до 40 кодируется ASCII 33-73
- '!' (ASCII 33) = Q0
- 'I' (ASCII 73) = Q40

**Phred+64 (старые версии Illumina):**
- Символ = chr(Q+64)
- ASCII 64-104

### 7.4 Заголовки FASTQ

Для Illumina обычно:

```
@<instrument>:<run>:<flowcell>:<lane>:<tile>:<x>:<y> <read>:<filtered>:<control>:<index>
```

### 7.5 Практическое задание

```python
import gzip
import matplotlib.pyplot as plt
import numpy as np

def read_fastq(filepath):
    opener = gzip.open if filepath.endswith('.gz') else open
    with opener(filepath, 'rt') as f:
        while True:
            header = f.readline().strip()
            if not header:
                break
            seq = f.readline().strip()
            plus = f.readline().strip()
            qual = f.readline().strip()
            yield header, seq, qual

def decode_phred(qual_str, offset=33):
    return [ord(c) - offset for c in qual_str]

# Анализ первых 10000 ридов
all_qualities = []
for i, (h, s, q) in enumerate(read_fastq("sample.fastq.gz")):
    if i >= 10000:
        break
    all_qualities.extend(decode_phred(q))

# График распределения качества
plt.hist(all_qualities, bins=range(41))
plt.xlabel("Phred score")
plt.ylabel("Count")
plt.show()

# Качество по позициям
max_len = 150
quals_by_pos = [[] for _ in range(max_len)]

for i, (h, s, q) in enumerate(read_fastq("sample.fastq.gz")):
    if i >= 10000:
        break
    quals = decode_phred(q)
    for pos, qv in enumerate(quals):
        if pos < max_len:
            quals_by_pos[pos].append(qv)

avg_qual = [np.mean(pos) if pos else 0 for pos in quals_by_pos]
plt.plot(avg_qual)
plt.xlabel("Position in read")
plt.ylabel("Average Phred score")
plt.show()
```

## Контрольные вопросы (расширенные)

**1. В чём принципиальное отличие метода Сэнгера от NGS?**

**Ответ:** Сэнгер секвенирует одну молекулу (результат ~ один рид), NGS требует клональной амплификации и читает миллионы молекул параллельно.

**2. Почему Сэнгер использует ddNTP, а Illumina — dNTP с терминаторами?**

**Ответ:** ddNTP не имеет 3'-OH, полимераза не может продолжить синтез (необратимая блокировка). Illumina использует обратимые терминаторы для циклического считывания.

**3. Какие проблемы решает клональная амплификация?**

**Ответ:** Одна молекула ДНК дает слишком слабый сигнал. Амплификация создает ~1000 копий в кластере, усиливая сигнал в 1000 раз.

**4. Разница между эмульсионной ПЦР, мостиковой амплификацией и RCA?**

**Ответ:** 
- Эмульсия: капли воды в масле, ПЦР в каплях
- Мостиковая: цепь гибридизуется с соседним олигонуклеотидом, образуя мост
- RCA: кольцевая матрица, полимераза много раз обходит кольцо

**5. Почему Ion Torrent не требует оптической системы?**

**Ответ:** Детектирует ионы H⁺ (pH) через электрохимический сенсор (CMOS), а не свет.

**6. Преимущества парных прочтений (paired-end)?**

**Ответ:** Лучше выравниваются, позволяют собирать геномы, определять структурные варианты, исправлять ошибки.

**7. Что такое гомополимеры и для каких платформ проблема?**

**Ответ:** Несколько одинаковых букв подряд (AAAA). Проблема для 454 и Ion Torrent (сложно считать длину).

**8. DNA nanoball и преимущества?**

**Ответ:** Конкатемер (RCA), гибридизуется с подложкой. Ошибки не накапливаются (всегда копируется с оригинала).

**9. Почему ONT читает очень длинные фрагменты?**

**Ответ:** ДНК протягивается через одну нанопору, длина зависит от целостности молекулы, не от химии.

**10. Как PacBio достигает высокую точность (HiFi)?**

**Ответ:** Циркулярное секвенирование — молекула зацикливается, полимераза много раз проходит, консенсус из множества прочтений.

**11. Что такое Phred score и кодирование?**

**Ответ:** Q = −10 log₁₀ P. Phred+33: Q кодируется chr(Q+33). Q30 = 99.9% точность.

**12. Почему качество падает к концу рида на Illumina?**

**Ответ:** Накопление ошибок фазирования (не все молекулы в кластере синхронны), фотоблидинг флуорофоров.

**13. Какие метаданные в заголовке FASTQ Illumina?**

**Ответ:** Прибор, номер запуска, ID проточной ячейки, лэйн, тайл, координаты x:y, номер рида, фильтрация, контроль, индекс.

**14. Когда имеет смысл Сэнгер до сих пор?**

**Ответ:** Плазмиды, небольшие фрагменты (< 1 kb), валидация мутаций, быстро и дешево для < 20 образцов.

**15. Какие технологии подходят для de novo сборки генома?**

**Ответ:** PacBio HiFi или ONT (длинные риды) + Illumina (коррекция). Длинные риды нужны для перекрытия повторов.
